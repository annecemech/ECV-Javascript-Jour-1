Introduction au Javascript - Jour 1

Vous allez maintenant utiliser Javascript sur une page web.
Pour cela, vous devrez écrire votre code Javascript dans le fichier script.js.

Comment ce fichier est lu par votre navigateur ? Inspectez le code de la page (clic droit > inspecter), vous trouverez une balise 'script' qui importe ce fichier juste à la fin du body (bonne pratique pour ne pas ralentir le chargement de la page et charger le script après les autres éléments).

Appuyez vous sur les slides du cours pour vous aider, ou directement sur la documentation, Javascript MDN.

Pour l'ensemble de ces exercices, n'oubliez pas d'y aller étape par étape et n'hésitez pas à utiliser des console.log et à regarder ce qui s'affiche dans la console du navigateur (Clic droit > Inspecter > Onglet console). Cela vous aidera à comprendre ce que vous manipulez.

Selectionner et manipuler des éléments dans le DOM

En javascript, sélectionnez le titre h1 de la page, changer sa couleur par "var(--purple)", puis remplacer le texte "Introduction au Javascript" par "Mes premiers pas en Javascript"

Utilisez document.querySelector pour venir sélectionner la balise h1 et stocker la dans une variable (const).

Venez ensuite manipuler cette variable pour changer le texte (innerText) et sa couleur (style et color).

En javascript, sélectionnez le bouton ci-dessous, changer sa couleur de fond par par "var(--pink)", la couleur du texte par "var(--dark-blue)" puis remplacer le texte "J'apprends le Javascript"

Utilisez document.getElementById pour venir sélectionner le bouton et stocker le dans une variable (const).

Venez ensuite manipuler cette variable pour changer le texte (innerText) et sa couleur (style et color) et sa couleur de fond (style et backgroundColor).
